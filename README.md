# Touring_Memory

## サイト概要
### サイトテーマ
自分がツーリングした記録を投稿<br>
他のユーザーと共有し、繋がりを増やすSNSサイト

### テーマを選んだ理由
カレンダー機能、外部API導入の練習のため<br>
自分自身、ツーリングしますがいつも1人なのでこのようなサイトがあれば、気軽にツーリング仲間を作れるのではないかと考えました。

### ターゲットユーザー
* バイクユーザー
* 1人ではなく誰かとツーリングを楽しみたい方

### 主な利用シーン
* 自身のツーリング予定の作成・確認
* ツーリング記録の作成
* バイクユーザーとのコミュニケーション
